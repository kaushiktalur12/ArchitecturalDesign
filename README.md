🏗️ Generative AI for Architectural Design

An AI-powered web application that automates the creation of architectural floor plans based on user requirements. The system generates optimized 2D floor plans and interactive 3D visualizations, helping users quickly explore different architectural layouts without requiring professional CAD knowledge.

---

📌 Overview

Generative AI for Architectural Design is a full-stack web application developed as an academic project. Users can enter plot dimensions, building type, room requirements, number of floors, design style, and building orientation. Based on these inputs, the application automatically generates multiple architectural floor plan variations along with interactive visualizations.

The project demonstrates how Artificial Intelligence can improve traditional architectural planning by reducing manual effort and providing faster conceptual designs.

🚀 Installation

Clone the repository

git clone https://github.com/your-username/Generative-AI-Architectural-Design.git

Move into the project folder

cd Generative-AI-Architectural-Design

Install Flask

pip install flask

Run the application

python app.py

Open your browser

http://127.0.0.1:5000
